Aetherius

Real-time Volumetric Cloud Generation Tool for Unity

Aetherius is a free tool developed for Unity's Built-In Render Pipeline that enables the creation and customization of fully volumetric traversable cloudscapes. It can create realistic, dynamic clouds on a planetary scale in a matter of minutes.
